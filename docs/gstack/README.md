# Gstack Workflow Artifacts

This repo tracks Codex-first gstack workflow outputs here.

- `brief.md` is the persisted `/office-hours` output
- `plan.md` is the persisted `/autoplan` output
- `retro.md` is the persisted `/retro` output

Machine-only workflow state belongs under `.codex-gstack/workflow/` and should stay untracked.
